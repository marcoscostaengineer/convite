Projeto: Convite Especial
Descrição: Projeto simples em PHP para enviar um convite e coletar números de WhatsApp.
Estrutura:
- index.php: HTML e visualização mínima (exibe mensagens).
- process.php: Lógica PHP para processamento do formulário.
- style.css: Estilos do projeto.
- script.js: Comportamento do formulário (exibição condicional).
- whatsapp_numbers.txt: Armazena os números de WhatsApp submetidos.
Instruções:
1. Coloque todos os arquivos no mesmo diretório no servidor.
2. Verifique se o servidor possui suporte a PHP.
3. Certifique-se de que a pasta possui permissão de escrita para o arquivo whatsapp_numbers.txt.
